# Arison N. — Portfolio

Personal portfolio website for Arison N., a Civil Engineering student and Interior Designer based in Hong Kong.

Built with [Astro](https://astro.build/).

## Getting Started

**Prerequisites:** Node.js >= 22.12.0

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

The site will be available at `http://localhost:4321`.

## Build & Preview

```bash
# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## Project Structure

```
src/
├── assets/          # Images and static assets
├── components/      # Reusable Astro components
├── data/            # Site data (navigation, etc.)
├── layouts/         # Page layouts
├── pages/           # File-based routing
├── scripts/         # Client-side JavaScript
└── styles/          # Global and component styles
public/              # Files served as-is (favicon, robots.txt)
```
